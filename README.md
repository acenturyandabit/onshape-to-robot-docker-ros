# Onshape to URDF with docker
## Requirements
- Docker installed
- Python installed
- Onshape model
- Onshape API key

## Steps
1. Copy `config.sample.json` to `config.json` and fill it in as per https://onshape-to-robot.readthedocs.io/en/latest/config.html
2. Run `makeURDF.py`
3. Collect output in `output`.
